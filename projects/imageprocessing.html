Image Processing

Summary

This Java project gave me the opportunity to revisit some image processing algorithms. A lot of the work here was studying the theory behind them, as well as tweaking parameters to create good line detection within a set of sample images.
The code is hosted on github here.

What I have achieved so far

  • Implemented a Gaussian filter that blurs the image by averaging out each pixel with the neighboring pixels
  • Implemented a generic convolution transformation, which applies a small matrix to each pixel in an image to get a new summed value for the pixel. This can be used to detect intensity and direction of edges (e.g. Robert's Cross filter)
  • Implemented a full Canny Edge detector, which uses a Gaussian filter and Sobel edge detector (convolution) to pull out edges. Then it uses "non-maximal suppression" to only take pixels that are considered at the top of the edge
  • Tweaked some parameters such as the size/deviation of the gaussian filter and changing the parameters on a high/low filter we apply on the image. This allowed us to get better image/edge outputs

Why?

  • Practicing Java as well as revisiting coding complex algorithms
  • Revising image processing and line detection algorithms, as the topic was quite interesting at university and has many exciting applications (e.g. medical, robotics, categorization of images)
  • It's quite a nice visual project, that you can actually show people the output of (examples to come!)
