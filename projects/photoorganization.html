Photo Organization

Summary

This project aimed to determine important features of some of my family photos, so we can simulate something like Amazon/Google photos. This involved determining which data is available and how we can extract those features from different photo types (IPhone, Android, scanned). Exif data allowed me to extract a significant amount of data, such that I could begin the catagorization process, but for scanned photos in particular, I wanted to investigate some machine learning techniques such as facial recognition to extract some more data about when and maybe where the photo was taken.

What I have achieved so far

  • Backing up the photos automatically to AWS bucket from multiple locations on my computer
  • Exif data extraction to determine location and date of the photo
  • Working with a using Python types to try to help with type safety inside the project
  • Initial investigation into facial recognition, using a pre-tuned face extractor, to get cropped faces from the images

Why?

  • Allows me to organize my photos for free.
  • Allows me to process the large backlog of my families photo libraries, especially from scanned images
  • Experiment with developing my own Python code from scratch using the newer type system, that can be used to improve code reliability
  • Play with facial recognition libraries in Python and developing my own classification

What's next

  • Classifying the cropped faces into the recognized people in the photos and adding that data to the algorithm
  • Combining multiple known classifications to fill in gaps or provide custom additional information (e.g. if it's my family in the photo in December then give it the tag Christmas)
  • Storing the data in a sqllite database rather than organizing into folders, to allow recatagorization of folders
  • Build a UI app to sort and filter the photos for display

Skills used

  • Python: project organization, Python types, photo Exif extraction, ML libraries
  • CV2: ML algorithm for classification of faces and how to apply different classification algorithms
